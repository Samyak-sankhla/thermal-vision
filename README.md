# Thermal Vision

Collection of tools for thermal image processing and analysis.

## Tools/Projects

### Intrinsic Calibration

Calibrate thermal camera intrinsic parameters using checkerboard patterns.

**Files:**
- `a4_checkerboard.py` - Generate checkerboard pattern for printing
- `camera_setup.py` - Capture thermal camera frames
- `calibrate.py` - Perform camera calibration/finds intrinsic parameters.

**Quick Start:**

Each module has its own dependencies. Install with:
```bash
cd Calibration/intrinsic
pip install -r requirements.txt
```

For detailed documentation, see [Calibration/intrinsic/README.md](Calibration/intrinsic/README.md)
